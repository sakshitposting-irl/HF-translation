# HF-translation
English to Hindi translator built by using a HuggingFace Pipeline

![Sample Input/Output](https://github.com/sakshitposting-irl/HF-translation/assets/134858442/8b099f82-2e71-4975-a573-b47d2ea25298)

The main aim of this translator was to understand the working of HF-pipelines and translation as a task in general.
The UI is hosted on Gradio.
